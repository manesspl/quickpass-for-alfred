Quick and simple: Type `pw` in Alfred to generate a strong password (Default length: 20 chars). You can specifiy the length of the password by passing an integer (for example: `12`). Hitting `âŽ Return` will copy the password to your clipboard.

## Installation
Clone this repository or download from the release page. Double click `Quickpass.alfredworkflow` and add the workflow to Alfred. 

## Requirements
- Alfred 5.x
- Python > 3.8
